보안을 위해, 해당 프로젝트에서는 S3 API에 접근하기 위해 API 키를 어플리케이션 활성화시 1회 요청합니다.
API 키는 저장되지 않습니다.


API 키를 만들려면 다음 과정을 따라합니다.
  1. AWS IAM 콘솔에 접속합니다.
  2. 사용자 생성을 누릅니다.
  3. 중복되지 않는 사용자명을 입력합니다. 해당 프로젝트에서는 s3_user로 생성하십시오.
  4. 직접 정책 연결를 선택하고, AmazonS3FullAccess를 검색합니다. 해당 권한은 발급된 키로 자신의 S3 API의 모든 접근을 허용하겠다는 의미입니다.
  5. 사용자 생성을 누릅니다.
  6. AWS IAM 콘솔에서 s3_user을 클릭합니다.
  7. 요약 창에서 푸른식 액세스 키 만들기를 클릭합니다.
  8. AWS 외부에서 실행되는 애플리케이션을 선택하고, 다음을 누릅니다.
  9. 액세스 키 만들기를 누릅니다.
  10. 출력된 액세스 키를 입력합니다. 경고: 한번 발급된 키는 페이지를 나가면 다시 볼 수 없습니다.
